Plataforma de Anúncios Automáticos

Carregue TXT (um email por linha) e/ou VCF. Configure abaixo a URL do backend (ex.: https://meu-backend.onrender.com).

Observações rápidas:
  • Se seu backend estiver em outro domínio, habilite CORS no servidor (permitir origem do GitHub Pages).
  • Defina a URL completa do backend no campo acima (sem barra final é ok). Ex: https://meu-backend.onrender.com
  • Use o botão Testar 1 Envio antes de iniciar envios em massa.
